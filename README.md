# Alex-held Tap

## How do I install these formulae?

`brew install alex-held/tap/<formula>`

Or `brew tap alex-held/tap` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
